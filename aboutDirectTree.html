Yahya Fahmi - DirectTree

DirectTree

DirectTree is a software application which which analyzes disk drives, identifying it's most storage consuming sectors.

The user-selected drive is scanned and stored in a Tree abstract data type (ADT), and once scanning is complete the user can traverse the drive and it's sub-directories.

With each traversal, a color-coded bar legend shows the most storage consuming items in the sub-directory, with the sections in the bar-legend split proportionally to item size.

The application's code and interface were created using the Qt C++ cross-platform framework, and the program is currently built for Windows OS.

Scanning Drive

The user has the choice to scan any hard disk drive or it's partition(s), external disk drives, or USB drives. Whichever drive is chosen will have all of it's sub-directories traveresed.

Traversing Directory

Traversing the scanned directory is very similar to the functionality of Windows Explorer, where the user can traverse each sub-directory.

Color-Coded Legend

The color-coded bar legend shows the the directory items with the largest sizes, with entry heights proportional to their size.
