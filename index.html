FILTERPLOT

FILTERPLOT is a software application which performs synthesis, simulation, and optimization of microwave radiofrequency filters.

The program was built and compiled using MATLAB R2020a but is still able to be ran by users without MATLAB installed.

FILTERPLOT was developed for the Centre of Integrated RF Engineering at the University of Waterloo.
