DirectTree

DirectTree is a software application which analyzes disk drives, identifying it's most storage consuming sectors.

A given drive that is scanned is stored in a Tree ADT, able to be traversed by the user.

The program is written with C++, with the interface built and compiled using the Qt cross-platform framework.
